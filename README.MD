# 🏡 Projeto Casas - Aluguéis

Plataforma desenvolvida para facilitar o processo de aluguel de casas, conectando proprietários e interessados de forma simples, rápida e eficiente.

O sistema permite buscar imóveis, visualizar detalhes, entrar em contato com anunciantes e gerenciar anúncios.

---

## 📌 Funcionalidades

- 🔍 Busca de casas para aluguel por cidade, bairro ou preço
- 🏠 Visualização detalhada do imóvel
- 📸 Galeria de fotos dos imóveis
- 📍 Localização integrada com mapa
- 👤 Cadastro e login de usuários
- 📝 Publicação de anúncios de casas
- ❤️ Favoritar imóveis
- 📞 Contato direto com o proprietário/anunciante

---

## 🎯 Objetivo

Facilitar o acesso a imóveis para aluguel, tornando o processo mais acessível e organizado tanto para quem busca quanto para quem anuncia.

---

## 🚀 Tecnologias Utilizadas

- Frontend: [React / HTML / CSS / JS]
- Backend: [Node.js / PHP / Python]
- Banco de Dados: [MySQL / MongoDB / PostgreSQL]
- API de Mapas: [Google Maps / OpenStreetMap]
